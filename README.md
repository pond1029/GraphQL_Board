# GraphQL을 사용한 게시판

GraphQL, JPA, Spring boot 를 사용하여 간단한 게시판 구현

상단에서 글을 작성하면 게시글 정보를 저장하고 게시글 리스트를 다시 조회한다.

삭제키를 입력하면 글을 삭제한다.

자바 1.8 버전 기준으로 개발되었으며, Linux, Window 환경에서 작동합니다.

**Chrome 브라우저에서 작동합니다.**
